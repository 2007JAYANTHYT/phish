How to Identify Phishing URLs

  • Check for misspellings or strange characters in the domain name
  • Look for HTTPS encryption (but note that some phishing sites use HTTPS too)
  • Be cautious of URLs that use IP addresses instead of domain names
  • Watch for URLs that are excessively long or contain unusual characters
  • Verify the sender's email address if the link came via email
  • Never enter personal information on sites you don't trust
